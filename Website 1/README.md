## üé® Design System


- **Serif**: Playfair Display (headings, elegance)
- **Sans-Serif**: Inter (body text, modern)

- Smooth cubic-bezier transitions (0.8s)
- Hover effects with lift transforms
- Image scaling on hover (1.05x - 1.15x)
- Shadow depth effects
